# EN-OS Builds

You can clone this repository and build any packet for EN-OS.
I will try create PKGBUILD file for all EN-Repo packages as quickly as possible

For build package:
*  `cd <folder with PKGBUILD>`
*  `makepkg -si`

### Packages list:
*  Blackarch-gtk (Blackarch gtk theme)
*  EN-Fastfetch (Fastfetch with EN-OS logo)
*  EN-Libpamac (Libpamac for EN-Pamac, dont have EN-OS modifications)
*  EN-Pamac (Pamac manager for EN-OS, have EN-OS modifications)
*  EN-OS-IceWM-Settings (IceWM settings with GUI)
*  EN-OS-Plymouth (Plymouth with EN-OS theme)
*  EN-OS-System-Manager (Start settings for EN-OS: driver manager, remote assistant creator, and more)
*  EN-OS-ZRam-manager (ZRam manager for EN-OS)
*  gnf-ttf (Font for EN-OS programs)
*  Postinstalls_icewm (Postinstall scripts for EN-OS IceWM edition)
*  Postinstalls_kde (Postinstall scripts for EN-OS KDE edition)
*  Visual-Studio-Code-bin (VS Code for Developers)

The list of packages may change over time.
